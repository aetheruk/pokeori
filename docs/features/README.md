# Feature Documentation

Complete documentation for all game features in Pokeori.

## Core Features
- [Battle System](/docs/features/battle-system.md) - Turn-based battles with advanced mechanics
- [Skills](/docs/features/skills.md) - Trainer, Explorer, and Researcher unlock guides
- [Pokedex](/docs/features/pokedex.md) - Track seen/caught Pokemon with research levels
- [Mini-Games](/docs/features/mini-games.md) - 18+ mini-games (slots, spelling, berry picker, etc.)
- [TCG System](/docs/features/tcg-system.md) - Trading Card Game collection and booster packs
- [Exploration](/docs/features/exploration.md) - Mine for items, discover Pokemon
- [Expeditions](/docs/features/expeditions.md) - Multi-step Explore routes and Chronicle story content
- [Voyages](/docs/features/voyages.md) - Send Pokemon on timed voyages for rewards
- [Tasks](/docs/features/tasks.md) - Daily and special research tasks
- [User State Storage](/docs/features/user-state-storage.md) - Split user-owned ledgers and sync payload strategy

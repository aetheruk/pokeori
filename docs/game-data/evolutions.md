# Evolutions Data

Evolution chains and methods in `src/data/evolutions/`.

The files under `src/data/evolutions/` are generated by `scripts/generate-evolutions.mjs`.
Do not patch generated evolution modules directly for durable changes; add source-data handling
or manual overrides to the generator instead.

Runtime target-form overrides live in `src/utilities/pokemon/evolution-targets.ts`. Pikachu normally evolves into base Raichu with a Thunder Stone, but Pikachu with `obtainedRegion` or `obtainedLocation` exactly set to `Alola` evolves into Alolan Raichu. Spewpa evolves into a randomly selected authored Vivillon form, including base and every pattern form in the Pokemon data. Rockruff evolves into base/Midday Lycanroc during day, Midnight Form during night, and Dusk Form during the dusk window. Toxel evolves into Amped/base Toxtricity or Low Key Toxtricity from its owned Pokemon nature. Darumaka evolutions roll 95% standard Darmanitan and 5% Zen Mode, with Galarian Darumaka using the matching Galarian standard or Zen form.

Runtime gender and time-of-day gates are enforced through `src/utilities/pokemon/evolution-conditions.ts`.
Generated `gender: 1` rows require female owned Pokemon and `gender: 2` rows require male owned Pokemon,
covering splits such as female Burmy into matching-cloak Wormadam, male Burmy into Mothim,
and female Combee into Vespiquen.
Time windows are day from 06:00-16:59, dusk from 17:00-17:59, and night from 18:00-05:59.

Regional source-form evolution overrides live in `scripts/generate-evolutions.mjs`. These keep
base, Alolan, Galarian, Hisuian, and Paldean source forms gated to the matching target form where
the species-level source data would otherwise collapse the rows together.

## Evolution Module Example
```typescript
export const evolutionChains = {
  "1": { // Bulbasaur
    chain: [1, 2, 3], // Species IDs
    methods: [
      { from: 1, to: 2, method: "level", level: 16 }, // Bulbasaur -> Ivysaur at 16
      { from: 2, to: 3, method: "level", level: 32 }  // Ivysaur -> Venusaur at 32
    ]
  }
}
```

## Evolution Methods
| Method | Description |
|--------|-------------|
| level | Evolve when reaching X level |
| stone | Use evolution stone |
| trade | Trade with another player |
| friendship | High friendship level |
| item | Use specific item |
| time | Day/night evolution |
| location | Evolve at specific location |

## Fields
| Field | Type | Description |
|-------|------|-------------|
| chain | number[] | Ordered species IDs in chain |
| methods | object[] | Evolution requirements |
| method | string | Evolution method |
| level | number | Required level (for level method) |
| item | string | Required item (for item method) |

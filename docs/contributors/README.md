# Contributor Guide

How to contribute to Pokeori.

## Guides
- [Code Standards](/docs/contributors/code-standards.md) - Coding conventions
- [Pull Request Guide](/docs/contributors/pull-request-guide.md) - PR process
- [Issue Templates](/docs/contributors/issue-templates.md) - Reporting bugs/features

## Getting Started
1. Fork the repository
2. Create a feature branch (`git checkout -b feature/your-feature`)
3. Make changes
4. Run lint (`pnpm run lint`)
5. Submit PR

## Code of Conduct
Be respectful to other contributors. No harassment or discriminatory language.

## Review Process
- All PRs require at least 1 review
- Tests must pass (once implemented)
- Lint check must pass
- Follow code standards

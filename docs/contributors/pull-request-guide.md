# Pull Request Guide

How to submit a PR for Pokeori.

## PR Checklist
- [ ] Code follows [Code Standards](/docs/contributors/code-standards.md)
- [ ] Lint passes (`bun run lint`)
- [ ] Tests added (if applicable)
- [ ] Documentation updated (if adding new feature)
- [ ] Changes tested manually

## PR Title Format
```
feat: add new mini-game
fix: correct battle damage calculation
docs: update API documentation
refactor: split large component
```

## PR Description Template
```markdown
## Description
Brief description of changes.

## Related Issue
Closes #123

## Changes Made
- Change 1
- Change 2

## Testing
Steps to test changes.

## Screenshots (if UI change)
Add screenshots here.
```

## Review Process
1. Maintainers will review within 48 hours
2. Address feedback promptly
3. Once approved, PR will be merged
